Agents MCP

How Smartsheet built a remote MCP server on AWS

出典: AWS Machine Learning Blog / 公開日: 2026-07-17 JST

読む価値: Smartsheet は remote MCP server を AWS 上に構築し、AI clients に製品データと操作を直接渡せる形にしました。token 量の削減と運用性の両方を狙っています。
