Agents Infra

Expanding Choice in Gemini Enterprise Agent Platform: Introducing Grounding with Parallel Web Search

出典: Google Developers Blog / 公開日: 2026-07-16 JST

読む価値: Gemini Enterprise Agent Platform に Parallel Web Systems の grounding provider が入ったことで、リアルタイムな web 事実に citations 付きで寄せられます。KYC、調査、更新確認のような用途に直結します。

開発者への示唆: grounding は「検索結果を使うか」ではなく、「どの provider を採用し、証跡をどう保持するか」という設計課題になります。

試すなら: 事実依存の agent で grounded response の citation を監査ログに残し、非 grounding 経路との差を比較してください。

Agents Workflow

Building scalable AI agents with modular prompt transpilation

出典: Google Developers Blog / 公開日: 2026-07-16 JST

読む価値: 単一の巨大 prompt を、依存関係付きの skill fragments に分けて build 時に検証する考え方です。失敗を runtime から CI に寄せられます。

開発者への示唆: agent の信頼性は prompt の長さではなく、モジュール化、静的検証、golden file の差分管理で決まります。

試すなら: system prompt を instructions、policy、tooling に分け、レンダリング結果の差分検査を CI に入れてください。

テーマ別メモ

  • grounding first: web grounding と citations が、agent の実運用で「正しい返答」の前提になっています。
  • prompt as build artifact: prompt は編集テキストではなく、検証可能な build artifact として扱う流れです。
  • review as controlled execution: review は comment 生成だけでなく、指示・環境・ネットワーク条件の制御まで含むようになっています。
